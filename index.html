About this Agent

Architecture, models & features

What is this?

A fully real-time, browser-based voice agent. You speak — it listens, thinks, and talks back. No button to hold. Voice Activity Detection (VAD) automatically detects when you start and stop speaking. Audio is processed through a concurrent async streaming pipeline — the LLM and TTS run in parallel, so you hear the first sentence while the rest is still being generated.

Features
Auto Voice Activity Detection (VAD)
No button to hold — detects when you start and stop speaking using real-time frequency analysis in the 300–3400 Hz speech band.
Concurrent Streaming Pipeline
LLM streams tokens sentence-by-sentence into a queue. TTS consumes each sentence in parallel — you hear the first sentence while the rest is still being generated.
Gapless Audio Playback
Uses Web Audio API to decode and schedule each sentence's audio buffer precisely back-to-back — no clicks or silence between sentences.
Live Transcript
Every exchange is shown in a chat UI — both your words and the agent's response appear in real-time.
Conversation Memory
Full conversation history is maintained in-session, so the agent understands context across multiple turns.
Customizable Agent
Change voice (ElevenLabs voice ID) and personality (system prompt) live from the Settings panel — no restart needed.
Multilingual
Saaras v3 supports Hindi, English, and code-switched Hinglish. The LLM responds in the same language you speak.
